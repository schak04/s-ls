# Debugging Lession Learnt (14-06-2026)

A crash often occurs far away from the actual bug.

#### My experience after debugging a segfault today (14-06-2026):
- Program crashed in `strcmp()`.
- The real bug was that the array being sorted was never populated.
- GDB revealed:
  - `arrSize = 6`
  - `idx = 0`
- AAAAND this led me directly to the root cause.

### For my future self

#### When debugging:
1. Find where the crash happens.
2. Inspect the data being used there.
3. Trace backwards to where that data came from.
4. Look for the first place reality differs from expectations.
